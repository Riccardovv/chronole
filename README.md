# CHRONO-LE

Adivina el año de eventos históricos.

## Cómo jugar
- Introduce un año (número entre -3000 y 2026)
- Recibe pistas de temperatura: 🔥 Muy caliente, 🌡️ Caliente, ❄️ Frío, 🧊 Muy frío
- Flecha ↑ (más tarde) o ↓ (más temprano) te indica la dirección
- 6 intentos máximo con pistas progresivas

## Modos
- 📅 Diario: Un evento al día
- ♾️ Infinito: Juega sin parar
- 🎯 Por categoría: Solo Guerras, Arte, Ciencia...

## Stack
- React + Vite
- CSS puro
- GitHub Pages
